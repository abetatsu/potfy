# Potfy.me

## Descriptions
GAOGAOゲート 卒業生向け企画 第一回「一緒にプロダクトを開発しよう」
ポートフォリオ掲載アプリ：「Potfy.me」

## How To Set Up This App

1. Run `cp .env.example .env`
2. Run `composer install`
3. Run `npm install`
4. Run `php artisan key:generate`
5. Run `php artisan migrate`


## Start server

For local,
```
php artisan serve
```

## Front-end Dev

- For development
```
npm run dev
```
```
npm run watch
```
